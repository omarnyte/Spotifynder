genreConnect

  • About genreConnect

    genreConnect is an audio visualization built using vanilla HTML, CSS, and JavaScript with a sprinke of jQuery for AJAX requests made to the Spotify API using an API wrapper.

  • Instructions

    To get started, simply begin typing the name of an artist. Please be aware that genreConnect can only search artists whose discography appears on Spotify.
