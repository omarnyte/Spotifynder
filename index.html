Welcome to Spotifynder


Enter an artist's name below to see their related artists. The graph will show you how many times each genre apears for those artists.
You can also search by clicking (and holding) the microphone icon and speaking speking the artist's name. You will have to enable browser access to your microphone when prompted.

    Spotifynder is an audio visualization built using vanilla HTML, CSS, and JavaScript with a sprinkle of jQuery (for an initial AJAX requests for a Spotify token) and an API wrapper (to fulfill subsequent API requests). The backend is served by Node.js utilizing the Express framework.


    The graphs and chartrs are built entirely on HTML and CSS, and they are made interactive with vanilla JavaScript.


    The layout for the project was inspired by histography.io
