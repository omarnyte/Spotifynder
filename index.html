Spotifynder is an audio visualization built using vanilla HTML, CSS, and JavaScript with a sprinke of jQuery (for an initial AJAX requests for a Spotify token) and an API wrapper (to fulfill subsequent API requests). The frontend is served by Node.js utilizing the Express framework.


The graphs are built entirely on HTML and CSS, and they are made interactive with vanilla JavaScript.
