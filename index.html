Welcome to Spotifynder


Enter an artist's name below to see their related artists. After selecting an artist, you can hover over the artist image to hear a preview of their music. The graph will show you how many times each genre appears for those artists.
You can also search by clicking (and holding) the microphone icon and speaking the artist's name. You will have to enable browser access to your microphone when prompted.
Consider using Chrome to search by voice!

    Spotifynder is an audio visualization built using vanilla HTML, CSS, and JavaScript with a sprinkle of jQuery (for an initial AJAX requests for a Spotify token) and an API wrapper (to fulfill subsequent API requests). The backend is served by Node.js utilizing the Express framework.


    The graphs and charts are built entirely on HTML and CSS, and they are made interactive with vanilla JavaScript. Voice recognition is provided by Google Chrome's native voice recognition.


    The layout for the project was inspired by histography.io
